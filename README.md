# mddb

A YAML + markdown card substrate for agentic and human knowledge work.

**Status: placeholder.** Real implementation is forthcoming. This release reserves the name on PyPI.

## Intent

`mddb` is a deliberately minimal substrate for storing knowledge as cards (YAML frontmatter + markdown body), queryable by agents and humans, with decades-durable on-disk format and git-backed history. It imposes no schema and privileges no fields — semantics live in layers above.
